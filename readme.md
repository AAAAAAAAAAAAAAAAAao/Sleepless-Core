\# Sleepless Core 🕹️



An original 2D psychological horror game featuring a tactical, turn-based combat system. Built from scratch using \*\*GameMaker Studio 2 (GML)\*\*.



> ⚠️ **Project Status:** Active Development (Pet-project)



\## 🌟 Key Features



\* **Psychological Horror Atmosphere:** Immersive narrative combined with custom-drawn pixel art sprites.

\* **Turn-Based Combat System:** A tactical combat loop built on precise timing, distinct character weights, and resource management.

\* **Custom AI Behavior:** Enemy entities utilize unique behavior trees and state-driven logic to challenge the player.

\* **Robust Code Architecture:** The game is built with a heavy emphasis on clean OOP principles and optimized performance.



\## 🛠️ Technical Deep Dive \& Architecture



As the sole developer of this project, I focused on writing scalable and crash-resistant code:

\* **State Machines (FSM):** Character states (idle, moving, attacking, hit) and enemy AI logic are separated using robust state machine patterns to ensure clean transitions and bug-free behavior.

\* **UI Logic \& Interaction:** Completely decoupled UI systems that handle dynamic data display and user input without impacting core game performance.

\* **Safe Error Handling:** Heavy focus on input validation and runtime error trapping to ensure the application remains stable under any unexpected user actions.

\* **Version Control:** Managed systematically using Git and GitHub Desktop with structured branching and clean commit history.



\## 🧰 Technologies Used



\* **Engine:** GameMaker Studio 2

\* **Language:** GameMaker Language (GML / OOP style)

\* **Tools:** Git, GitHub Desktop, Pixel Art Editors
